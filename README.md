# HyperOS3-Debloat

HyperOS3 专用精简模块，支持 4 种精简模式，安全移除不必要的系统应用。

## 适用环境

- 系统：HyperOS3
- 框架：KernelSU / Magisk

## 安装方法

1. 从 [Releases](https://github.com/wenrouhao/HyperOS3-Debloat/releases) 下载最新版本 ZIP
2. 打开 KernelSU 管理器 → 模块 → 从本地安装 → 选择下载的 ZIP
3. 等待安装完成，**不要重启**
4. 按照屏幕提示操作（见下方按键说明）
5. 操作完成后重启

## 按键说明

安装过程中需要使用音量键进行选择：

| 按键 | 作用 |
|------|------|
| **音量+** | 确认当前选项 |
| **音量-** | 切换到下一个选项 |

选项会循环显示：`1 → 2 → 3 → 4 → 1...`

## 精简模式

### 模式 1：快速精简（推荐）

精简广告、推送、游戏、无障碍相关，保留所有核心功能。

**适合：** 大部分用户，安全无风险

### 模式 2：标准精简

在快速精简基础上，额外精简小爱视觉翻译、云服务、跨屏协同、汽车互联、互联互通、应用商店。

**适合：** 不用小米云服务、汽车互联的用户

### 模式 3：深度精简

在标准精简基础上，额外精简系统服务、窗口管理、AI 引擎、system_ext 服务。

**适合：** 追求极致精简的用户，可能影响部分系统功能

### 模式 4：自定义模式

每组功能单独选择，自由度最高。每组有三个选项：

| 选项 | 作用 |
|------|------|
| **全部精简** | 精简该组全部应用 |
| **保留核心** | 只精简低频应用，保留高频核心功能 |
| **全部保留** | 不精简该组任何应用 |

**自定义模式包含以下分组：**

| 分组 | 包含应用 | 保留核心 |
|------|---------|---------|
| AI/小爱 | 小爱推荐+超级小爱+语音唤醒+小爱视觉翻译+小爱服务 | 超级小爱+语音唤醒 |
| 搜索/游戏 | 全局搜索+游戏服务+小游戏服务 | 全局搜索 |
| 无障碍/宏 | 无障碍开关+宏+GPU驱动更新 | 无障碍开关 |
| 系统预装 | 下载管理+游戏中心+扫一扫 | 下载管理 |
| 云备份 | 小米云服务+小米云同步+小米云备份 | 小米云服务 |
| 跨屏协同 | 小米妙享 | 小米妙享 |
| 汽车互联 | 小米汽车互联 | 小米汽车互联 |
| 互联互通 | 互联互通+跨设备通信+小米互联服务 | 互联互通+小米互联服务 |
| 澎湃AI引擎 | 小米澎湃AI引擎 | 小米澎湃AI引擎 |
| 应用商店 | 小米应用商店 | 小米应用商店 |
| 负一屏/弹幕 | 智能助理+弹幕通知+内容扩展 | 智能助理 |
| 窗口管理/报告/注册 | 窗口管理+UI报告+注册服务 | 窗口管理 |
| 系统服务 | 质量监控+系统守护+虚拟SIM+紧急信息+功耗分析+系统助手 | 质量监控+系统守护 |

## 危险项

选择完模式后，会额外询问以下危险项（可跳过）：

| 危险项 | 风险 |
|--------|------|
| NFC 服务 | 门禁/公交卡/付款可能失效 |
| 系统安全组件 | 可能影响应用兼容性 |
| 搜狗输入法 | 请确保已安装其他输入法 |
| 讯飞输入法小米版 | 请确保已安装其他输入法 |

## 更新方法

直接覆盖刷入新版本 ZIP 即可，无需卸载旧版本。

## 卸载方法

在 KernelSU 管理器中卸载模块，重启即可恢复。
