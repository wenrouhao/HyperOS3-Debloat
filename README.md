# HyperOS3-Debloat

HyperOS3 专用精简模块，支持 4 种精简模式，安全移除不必要的系统应用。

## 适用环境

- 系统：HyperOS3
- 框架：KernelSU / Magisk

## 安装方法

1. 从 [Releases](https://github.com/wenrouhao/HyperOS3-Debloat/releases) 下载最新版本 ZIP
2. 打开 KernelSU 管理器 → 模块 → 从本地安装 → 选择下载的 ZIP
3. 等待安装完成，**不要重启**
4. 按照屏幕提示操作（见下方按键说明）
5. 操作完成后重启

## 按键说明

安装过程中需要使用音量键进行选择：

| 按键 | 作用 |
|------|------|
| **音量+** | 确认当前选项 |
| **音量-** | 切换到下一个选项 |

选项会循环显示：`1 → 2 → 3 → 4 → 1...`

## 精简模式

### 模式 1：快速精简（推荐）

精简广告、推送、游戏、无障碍相关，保留所有核心功能。

**适合：** 大部分用户，安全无风险

### 模式 2：标准精简

在快速精简基础上，额外精简云服务、负一屏、汽车互联、互联互通、应用商店。

**适合：** 不用小米云服务、汽车互联的用户

### 模式 3：深度精简

在标准精简基础上，额外精简系统服务、窗口管理、AI 引擎、system_ext 服务。

**适合：** 追求极致精简的用户，可能影响部分系统功能

### 模式 4：自定义模式

每组功能单独选择保留或精简，自由度最高。

**操作方式：**
- 每组会显示 `音量+: 保留xxx` 和 `音量-: 全部精简xxx`
- 按音量+保留该组的核心功能
- 按音量-精简该组全部内容

**自定义模式包含以下分组：**

| 分组 | 保留核心（音量+） | 全部精简（音量-） |
|------|------------------|------------------|
| AI/小爱模块 | 保留语音+唤醒 | 全部精简 |
| 搜索/游戏 | 保留搜索框 | 全部精简 |
| 无障碍/宏 | 保留无障碍服务 | 全部精简 |
| 系统预装 | 保留下载管理 | 全部精简 |
| 云备份 | 保留云服务基础 | 全部精简 |
| 跨屏协同 | 保留基础投屏 | 全部精简 |
| 汽车互联 | 保留基础连接 | 全部精简 |
| 互联互通 | 保留基础连接 | 全部精简 |
| 澎湃AI引擎 | 保留基础AI服务 | 全部精简 |
| 应用商店 | 保留基础商店 | 全部精简 |
| 负一屏/弹幕 | 保留小组件/负一屏基础 | 全部精简 |
| 窗口管理/报告/注册 | 保留窗口管理 | 全部精简 |
| system_ext服务 | 保留MiSight+Daemon | 全部精简 |

## 危险项

选择完模式后，会额外询问以下危险项（可跳过）：

| 危险项 | 风险 |
|--------|------|
| NFC 服务 | 门禁/公交卡/付款可能失效 |
| 系统安全组件 | 可能影响应用兼容性 |
| 搜狗输入法 | 请确保已安装其他输入法 |

## 更新方法

直接覆盖刷入新版本 ZIP 即可，无需卸载旧版本。

## 卸载方法

在 KernelSU 管理器中卸载模块，重启即可恢复。
