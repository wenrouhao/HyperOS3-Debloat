# HyperOS3-Debloat

HyperOS3 专用精简模块，支持 4 种精简模式 + WebUI 管理，安全移除不必要的系统应用。

## 适用环境

- 系统：HyperOS3
- 框架：KernelSU / Magisk

## 功能特性

- 🎛️ 4 种精简模式（快速/标准/深度/自定义）
- 📱 WebUI 管理界面（暗色/亮色主题、搜索、分组折叠）
- ⚠️ 危险项精简确认弹窗
- 🔄 覆盖安装自动保留配置
- 📦 73 个应用支持精简

## 安装方法

1. 从 [Releases](https://github.com/wenrouhao/HyperOS3-Debloat/releases) 下载最新版本 ZIP
2. 打开 KernelSU 管理器 → 模块 → 从本地安装 → 选择下载的 ZIP
3. 等待安装完成，**不要重启**
4. 按照屏幕提示操作（见下方按键说明）
5. 操作完成后重启

## 按键说明

安装过程中需要使用音量键进行选择：

| 按键 | 作用 |
|------|------|
| **音量+** | 确认当前选项 |
| **音量-** | 切换到下一个选项 |

选项会循环显示：`1 → 2 → 3 → 4 → 1...`

## WebUI 管理

重启后在 KernelSU 管理器中打开模块，点击 WebUI 即可：

- 🔍 搜索应用名或包名
- 🌙 暗色/亮色主题切换（跟随系统或手动）
- 🎚️ 逐个应用开关精简
- ⚠️ 危险项开启时弹出警告确认
- 💾 保存配置并重启生效

## 精简模式

### 模式 1：快速精简（推荐）

精简广告追踪、游戏中心相关，保留所有核心功能。

**适合：** 大部分用户，安全无风险

**精简内容：**
- 广告追踪：智能服务（MSA）、安全追踪服务、AnalyticsCore、广告隐私权（apex）
- 游戏中心：游戏服务SDK、小游戏服务、游戏中心、游戏高能时刻

### 模式 2：标准精简

在快速精简基础上，额外精简设备互联/生态相关。

**适合：** 不用小米云服务、跨设备互联的用户

**新增精简：**
- 设备互联/生态：小米云服务、小米云同步、小米云备份、小米妙享、小米汽车互联、小米汽车互联服务、互联互通、跨设备通信、互联通信服务、数字钥匙、骑行模式

### 模式 3：深度精简

在标准精简基础上，额外精简AI/小爱、内容/工具/安全、系统服务、无障碍/宏。

**适合：** 追求极致精简的用户，可能影响部分系统功能

**新增精简：**
- AI/小爱：小爱推荐、超级小爱、语音唤醒、小爱视觉翻译、小爱服务、澎湃AI引擎
- 内容/工具/安全：智能助理、弹幕通知、内容扩展、小米画报、小米云盘、用户反馈、快应用服务、黄页、三方应用异常分析、应用扩展服务、悬浮球、家人守护、小米音乐
- 系统服务：下载管理、扫一扫、系统打印服务、打印处理服务、工作设置、窗口管理、UI报告、注册服务、质量监控、系统守护、虚拟SIM、紧急信息、功耗分析、系统助手、网络位置服务、健康数据共享（apex）、全局搜索、SIM卡应用、运营商通知、通话记录备份、谷歌后台更新、日志抓取、触摸轨迹服务
- 无障碍/宏：无障碍开关、宏、GPU驱动更新

### 模式 4：自定义模式

每组功能单独选择，自由度最高。可逐个应用选择精简或保留。

**交互流程：**
1. 选择自定义模式
2. 逐个分组选择（音量+精简此组，音量-进入选择）
3. 进入分组后，逐个应用选择（音量+精简，音量-保留）
4. 最后确认精简列表

**自定义模式包含以下分组：**

| 分组 | 包含应用 |
|------|---------|
| AI/小爱 | 小爱推荐、超级小爱、语音唤醒、小爱视觉翻译、小爱服务、澎湃AI引擎 |
| 游戏中心 | 游戏服务SDK、小游戏服务、游戏中心、游戏高能时刻 |
| 设备互联/生态 | 小米云服务、小米云同步、小米云备份、小米妙享、小米汽车互联、小米汽车互联服务、互联互通、跨设备通信、互联通信服务、数字钥匙、骑行模式 |
| 广告/追踪 | 智能服务（MSA）、安全追踪服务、AnalyticsCore、广告隐私权（apex） |
| 内容/工具/安全 | 智能助理、弹幕通知、内容扩展、小米画报、小米云盘、用户反馈、快应用服务、黄页、三方应用异常分析、应用扩展服务、悬浮球、家人守护、小米音乐 |

## 危险项

选择完模式后，会额外询问以下危险项（可跳过）：

| 危险项 | 风险 |
|--------|------|
| NFC 服务 | 门禁/公交卡/付款可能失效 |
| 系统安全组件 | 可能影响应用兼容性 |
| 搜狗输入法 | 请确保已安装其他输入法 |
| 讯飞输入法小米版 | 请确保已安装其他输入法 |
| Cell Broadcast Service | **紧急警报服务**，精简后收不到地震/灾害预警 |
| 小米智能卡 | **NFC核心组件**，精简后门禁/公交卡/付款全部失效 |
| 米币支付 | 小米支付服务，精简后米币相关功能失效 |
| 小米应用商店 | 应用下载，精简后无法通过官方商店下载应用 |

## 更新方法

直接覆盖刷入新版本 ZIP 即可，无需卸载旧版本。

## 卸载方法

在 KernelSU 管理器中卸载模块，重启即可恢复。
